Introduction

When setting up a live event a process is created that sends information to your OTT apps about the state of the live event. The events are:

  • Reminder of coming live event
  • Live event delayed
  • Event is live
  • Replay available soon (VOD being created from live event)
  • Live event ended (with VOD recording available)
  • Live event ended (no VOD recording available)
  • Live event cancelled

The following section of this document shows example app pages from a Beacon web app. In the last section are app pages from a subscription process, which makes use of some of the previously mentioned live event state app pages.
