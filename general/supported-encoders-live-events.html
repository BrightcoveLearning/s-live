Supported Encoders for Live Events

This topic is intended to provide some guidelines for hardware encoder settings when conducting a live stream event. For a list of all the options available from Brightcove for delivering live streaming video, see Delivering Live Streams.

The following is a list of encoders that are generally known to work with Brightcove Live. Other encoders may also work; this list contains the encoders that we have specifically tested.

Also note that supported does not mean that every feature is supported. Specific capabilities (such as sending cuepoints) vary among different encoders.
  • XSplit
  • OBS
  • Wirecast
  • Cambria Live
  • Liveshell
  • AWS Elemental Live
  • Haivision
  • LiveU
  • Matrox
  • NewTek
  • Teradek

If your encoder is not listed, use SRT instead of RTMP as the transport protocol if RTMP doesn't work/connect. SRT has several benefits over RTMP in terms of reliability and observability. If it works, then it can be used at risk.

However, if using SRT doesn't work, we suggest switching to another encoder on the list of supported encoders to avoid incompatibility.
