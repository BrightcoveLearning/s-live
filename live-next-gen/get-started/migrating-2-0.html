Overview

Live 2.0 is a new live-streaming solution that is reliable, consistent, and scalable.

The objectives for Live 2.0 are:

  • Improved System Reliability
  • Deliver key features that are required
  • Live streaming foundation that is extensible for the future

To create API credentials, go to Managing API Authentication Credentials.

To find more information about APIs, see API reference.

Navigating to Live 2.0

  1. Navigate to the Live Module.
  2. Click Switch to Live 2.0.

Creating a New Channel

Note: If you are coming from the Live 1.0 documentation, please notice that Live 2.0 uses the terms Linear Channel for what it used to call Channel and Event Channel for what it used to call Event. The terminology jobs will no longer be used in Live 2.0.
  1. Navigate to the Live Module.
  2. Click Create Channel.

    Typically, event channels are created to stream shorter-length, one-time, or recurring events. Linear channels are typically used when delivering 24/7 live events. Linear channels and events channels are billed differently - check with your Customer Success manager for details.

  3. Fill your Live Channel information.

    For more information about configuring your Live Channel, see Creating a new event.

    In this step, you can activate Redundant Channels if needed.
  4. Click Save.
  5. Click Start Streaming to obtain the Input URL. The channel will start, and the URL will be generated. This may take more than a minute.
  6. Find your Encoder source URL.
    Note: Notice that for the RTMP streams, the Stream name/key for Live 2.0 channels is blive.

    You can find your Source URL once your Channel is Live. It can take a few minutes for the Channel to Start.
  7. Stop your Live Channel by clicking Stop Streaming.

Editing a Channel

  1. Navigate to the Live Module.
  2. Click (...) next to your live channel.
  3. Click Edit.

  4. Edit your JSON file and click Update Live Channel

When a channel is in active status, actions such as Edit and Delete will be greyed out and unavailabe untill streaming is stopped.

Deleting a Channel

  1. Navigate to the Live Module.
  2. Click (...) next to your live channel.
  3. Click Delete.

Note that when you click (...), the following options also appear:

  • Duplicate - Duplicate makes it easy to create another channel with the same settings that were used.
  • Start - Start makes to run the stream selected.
  • Stop - Stop makes a stream stop when it is running.
  • Edit - Click edit to view/edit the Channel Edit - JSON (only available when streaming is stopped).
  • Live Media Asset - Click Live Media Asset to view the live event video properties.
